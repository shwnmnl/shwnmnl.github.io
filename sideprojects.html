SCP AutoArchivist

The AutoArchivist results from the convergence of my interests in eldritch horror (e.g. The Magnus Archives podcast) and data science. This project is part of the SCP Foundation, a fictional organization tasked with securing, containing and protecting anomalous phenomena; as well as a wiki-style online collaborative writing project. With no clear order in which to read anything, I decided to code a simple Python app that sends a random SCP item by email daily, that eventually grew to what is is now.

OpenProjection

OpenProjection is an open-source machine learning project designed in the spirit of translational science. Its main use is as a beginner friendly tutorial for machine learning in Python using psychological data. It covers different types of Supervised Learning problems, including both regression and classification, as well as an embedded Unsupervised Learning problem. The entire thing can be run from start to finish in Google Colab.
